# 🔍 ГДЕ НАЙТИ ПОЛЕ NAME в n8n

## 📍 ПОШАГОВО - где искать поле Name:

### Когда вы создаете Postgres credential, экран выглядит так:

```
┌─────────────────────────────────────────────────────┐
│  Postgres credential                            [X] │ ← Заголовок окна
├─────────────────────────────────────────────────────┤
│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │ Name *                                      │   │ ← ВОТ ОНО! Самое первое поле!
│  │ [_________________________________]         │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │ Host *                                      │   │
│  │ [_________________________________]         │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │ Database *                                  │   │
│  │ [_________________________________]         │   │
│  └─────────────────────────────────────────────┘   │
│                                                     │
│  ... (остальные поля)                              │
│                                                     │
│                    [Test] [Save]                    │ ← Кнопки внизу
└─────────────────────────────────────────────────────┘
```

---

## 🎯 ПОЛЕ NAME:

**Это САМОЕ ПЕРВОЕ поле** в форме, находится:
- **В САМОМ ВЕРХУ** после заголовка "Postgres credential"
- **ПЕРЕД** полем "Host"
- Обычно помечено звездочкой: **Name \***
- Это текстовое поле для ввода

---

## ⚠️ ЕСЛИ ВЫ НЕ ВИДИТЕ ПОЛЕ NAME:

### Вариант 1: Прокрутите форму вверх
- Возможно форма прокручена вниз
- Прокрутите колесиком мыши ВВЕРХ
- Поле Name должно быть в самом начале

### Вариант 2: Форма открыта не полностью
- Закройте окно (крестик)
- Повторите заново:
  - Credentials → Add Credential → Postgres
- Убедитесь что окно открылось полностью

### Вариант 3: У вас старая версия n8n
- В некоторых версиях поле называется по-другому
- Ищите поле с названиями:
  - **"Credential Name"**
  - **"Display Name"**
  - **"Label"**

---

## 🆘 ЧТО ДЕЛАТЬ ПРЯМО СЕЙЧАС:

1. **Сделайте скриншот** всей формы которую вы видите
2. Или **опишите**, какие поля вы видите по порядку сверху вниз
3. Или просто **перечислите** все названия полей которые видите

Тогда я точно скажу где Name!

---

## ✅ АЛЬТЕРНАТИВНЫЙ СПОСОБ (если совсем не найдете):

Если поле Name не находится:

1. **Оставьте его пустым** (если можно)
2. Или введите в первое попавшееся текстовое поле: **"Apallaktis PostgreSQL"**
3. Заполните обязательные поля:
   - Host: `postgres`
   - Database: `apallaktis`
   - User: `apallaktis`
   - Password: `apallaktis_password_change_me`
   - Port: `5432`
4. Нажмите **Save**
5. Credential сохранится и будет работать
