# ЗАКОНЫ ДИЗАЙНА ПРОЕКТА

## ЗАКОН КНОПОК-ЗАГОЛОВКОВ
Применяется к кнопкам типа: "Добавить расход", "Способы оплаты", "Объекты", "Личный кабинет", "Управление категориями" и т.д.

**Эталон**: страница с 4 кнопками (page-pay)

### Параметры:
1. **Контейнер**:
   - `paddingLeft: '40px'`
   - `paddingRight: '40px'`

2. **Ширина кнопки**:
   - `w-full` (100% от контейнера)

3. **Высота**:
   - `minHeight: '52px'`

4. **Шрифт**:
   - `text-button` класс

5. **Расстояние между кнопками**:
   - `gap-12` (48px) или `marginBottom: '48px'`

## ЗАКОН ДЛЯ КНОПОК ЭЛЕМЕНТОВ СПИСКОВ
Применяется к карточкам объектов, расходов, способов оплаты внутри списков.

### Параметры:
1. **Ширина**:
   - `w-full` (из закона кнопок-заголовков)

2. **Шрифт**:
   - Размеры из `text-button`

3. **Цвет**:
   - По дизайну страницы

4. **Высота**:
   - Автоматическая, зависит от содержимого
   - НЕ фиксированная!

## ЗАКОН ОТСТУПОВ В ПОЛЯХ ВВОДА
1. **Отступ слева и справа внутри поля**:
   - `paddingLeft: '40px'`
   - `paddingRight: '40px'`

2. **Цвет текста**:
   - `color: 'var(--deep-teal)'`

## ЗАКОН ДЛЯ ИНТЕРНАЦИОНАЛИЗАЦИИ
1. **Слова которые НЕ переводятся**:
   - email
   - VIP
   - Voice
   - WhatsApp
   - Viber
   - Referral Program

2. **Первая буква заглавная, остальные строчные** для всех заголовков кнопок
