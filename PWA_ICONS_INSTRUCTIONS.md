# Инструкция по созданию иконок для PWA

## Требуемые иконки

Для полноценной работы PWA необходимо создать следующие иконки:

### 1. **icon-192.png** (192x192px)
- Формат: PNG
- Размер: 192x192 пикселей
- Назначение: Стандартная иконка приложения для Android

### 2. **icon-512.png** (512x512px)
- Формат: PNG
- Размер: 512x512 пикселей
- Назначение: Иконка высокого разрешения для Android splash screen

### 3. **icon-maskable-192.png** (192x192px)
- Формат: PNG
- Размер: 192x192 пикселей
- Назначение: Maskable иконка для адаптивной формы (круг, квадрат, squircle)
- **Важно**: Основной контент должен быть в центральной области (80% от размера)

### 4. **icon-maskable-512.png** (512x512px)
- Формат: PNG
- Размер: 512x512 пикселей
- Назначение: Maskable иконка высокого разрешения

## Дизайн иконки

### Цвета бренда ΑΠΑΛΛΑΚΤΗΣ:
- Основной цвет: `#01312d` (deep-teal) - темно-бирюзовый
- Акцентный цвет: `#ff8f0a` (slogan-color) - оранжевый
- Дополнительный: `#b1d1a2` (zanah) - светло-зеленый
- Фон: `#daf3f6` (polar) - светло-бирюзовый

### Концепция дизайна:
1. **Вариант 1**: Буква "Α" (альфа) на темно-бирюзовом фоне
   - Буква оранжевого цвета (#ff8f0a)
   - Фон темно-бирюзовый (#01312d)

2. **Вариант 2**: Логотип с домиком и галочкой
   - Символизирует завершенные объекты
   - Минималистичный дизайн

3. **Вариант 3**: Абстрактная иконка с документом и евро (€)
   - Отражает суть приложения (финансы)

## Как создать maskable иконки

Maskable иконки должны иметь **safe zone** в центре:

```
┌──────────────────────┐
│                      │  ← Padding zone (10%)
│   ┌──────────────┐   │
│   │              │   │
│   │  Safe zone   │   │  ← Main content (80%)
│   │   (logo)     │   │
│   │              │   │
│   └──────────────┘   │
│                      │  ← Padding zone (10%)
└──────────────────────┘
```

### Тестирование maskable иконок:
- Используйте **Maskable.app** (https://maskable.app/)
- Проверьте, как иконка выглядит в разных формах (круг, квадрат, squircle)

## Пример создания через Figma

### Шаг 1: Создание артборда
- Создать новый файл в Figma
- Добавить Frame размером 512x512px
- Назвать "icon-512"

### Шаг 2: Дизайн
- Добавить прямоугольник 512x512px с цветом #01312d
- Добавить текст "Α" (буква альфа)
- Размер шрифта: 350px
- Цвет: #ff8f0a
- Выровнять по центру

### Шаг 3: Экспорт
- Выбрать Frame
- Export → PNG → 1x (512x512)
- Сохранить как `icon-512.png`

### Шаг 4: Создать другие размеры
- Дублировать Frame
- Изменить размер на 192x192px
- Экспортировать как `icon-192.png`

### Шаг 5: Maskable версии
- Добавить padding 10% вокруг основного контента
- Экспортировать как `icon-maskable-512.png` и `icon-maskable-192.png`

## Размещение иконок

После создания поместите все иконки в папку:
```
frontend/public/
  ├── icon-192.png
  ├── icon-512.png
  ├── icon-maskable-192.png
  └── icon-maskable-512.png
```

## Проверка

После размещения иконок:
1. Запустите dev сервер: `npm run dev`
2. Откройте Chrome DevTools → Application → Manifest
3. Проверьте, что все иконки загружаются
4. Запустите Lighthouse audit для проверки PWA score

## Примечание

**ВРЕМЕННОЕ РЕШЕНИЕ**: Пока иконки не созданы, можно использовать placeholder иконки.
Создайте простой SVG файл с буквой "Α" и конвертируйте в PNG через:
- https://svgtopng.com/
- https://cloudconvert.com/svg-to-png
